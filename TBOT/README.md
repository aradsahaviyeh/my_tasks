## Telegram Bot Setup
## راه اندازی ربات تلگرام

1. ساخت ربات با BotFather و دریافت توکن
2. ساخت فایل .env و قرار دادن توکن در آن
3. ساخت و فعال‌سازی virtual environment
4. ساخت فایل .env و قرار دادن توکن در آن
5. نصب وابستگی‌ها:
   pip install -r requirements.txt
6. اجرای ربات:
   python bot.py

.پس از اجرای ربات، می‌توانید با ارسال پیام به ربات، پاسخ دریافت کنید
