## Django Setup
## راه اندازی جنگو

1. ساخت و فعال‌سازی virtual environment
2. نصب وابستگی‌ها:
   pip install -r requirements.txt
3. اجرای پروژه:
   python manage.py runserver
4. رفتن به آدرس:
    yourhost:port/web/messages
